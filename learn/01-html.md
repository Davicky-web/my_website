# HTML Basics

HTML (HyperText Markup Language) gives structure to web pages. Key concepts:

- Elements: `<h1>`, `<p>`, `<a>`, `<img>`, `<ul>`, `<form>`
- Attributes: `href`, `src`, `alt`, `class`, `id`
- Semantic tags: `<header>`, `<main>`, `<section>`, `<article>`, `<footer>`

Quick exercise

1. Open `index.html`.
2. Add a new `<section>` with an `<h2>` and a paragraph describing yourself.
3. Add an image using `<img src="path/to/image.jpg" alt="desc">`.
