# JavaScript Basics

JavaScript adds behavior to web pages. Core ideas:

- DOM selection: `document.getElementById`, `querySelector`
- Events: `click`, `submit`, `DOMContentLoaded`
- Fetch API: `fetch()` to call servers
- Local state: `localStorage` to persist settings

Quick exercise

1. Open `script.js` and find the theme toggle code.
2. Modify the icon or add console.log statements to see it run.
3. Add a small function that updates the page title after 3 seconds.
