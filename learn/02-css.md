# CSS Basics

CSS (Cascading Style Sheets) controls presentation. Key topics:

- Selectors: `.class`, `#id`, `element`
- Box model: `margin`, `border`, `padding`, `width`
- Layout: Flexbox (`display:flex`) and Grid
- Responsive: media queries `@media (max-width: 640px){...}`

Quick exercise

1. Open `styles.css`.
2. Change the `--accent` color in `:root` and see the buttons update.
3. Make the `.hero` text center-aligned for small screens using a media query.
