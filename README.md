# Simple Site

This is a minimal static website you can open locally or serve with a dev server.

Quick start

1. Open `c:/Users/Glory/vscode/my-website` in VS Code.
2. Open `index.html` and preview using the Live Server extension, or open the file in a browser.

Commands (optional)

```bash
cd "c:/Users/Glory/vscode/my-website"
# initialize git (optional)
git init
git add .
git commit -m "Initial site"
```

To deploy: push this folder to GitHub and enable GitHub Pages, or drag the folder into Netlify/Vercel.

Learn and extend

- See the short lessons under the `learn/` folder: `learn/01-html.md`, `learn/02-css.md`, `learn/03-js.md`.
- Deployment instructions are in `DEPLOY.md`.
